# People 360

People 360 is a modern SaaS platform designed for interactive safety training. This project is a frontend prototype showcasing a high-converting landing page for the application.

## Features Highlighted

- **Interactive Courses:** Access OSHA and ISO compliant video courses.
- **Automated Reporting:** Generate compliance reports in real-time.
- **Smart Alerts:** Automated notifications for upcoming renewals and incomplete training.
- **Mobile-First Learning:** Training accessible on-the-go from any device.

## Technologies Used

- HTML5
- CSS3 (Vanilla, custom design system)
- JavaScript (Vanilla)
- Phosphor Icons

## Getting Started

To view the project locally, simply clone the repository and open the `index.html` file in your preferred web browser.

```bash
git clone https://github.com/clederamarkx-cody/sto-draft-1.git
cd sto-draft-1
```

Then, open `index.html`. No build steps or local servers are required for this static prototype.
